# 课程管理系统 java1162

## 项目概述

课程管理系统是一款基于前后端分离架构的Web应用程序，旨在提供高效便捷的课程管理功能。系统支持管理员、教师和学生三种角色，功能全面，操作简便。

## 技术栈

- **后端技术**：Java、SpringBoot、Mysql
- **前端技术**：Vue
- **开发工具**：IDEA、JDK1.8、Maven
- **系统环境**：Windows
- **数据库**：Mysql8.0

## 功能模块

### 系统角色

- 管理员
- 教师
- 学生

### 管理员模块

- 首页
- 系管理
- 专业管理
- 办理管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

### 教师模块

- 首页
- 授课查询
- 教师课表
- 成绩查询

### 学生模块

- 首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护

## 运行环境

- 后端：JDK1.8及以上版本，Mysql数据库8.0及以上版本
- 前端：现代浏览器支持
